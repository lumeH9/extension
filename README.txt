Program description/motivation

Program that keeps track of the time you spend on different websites. It will be integrated into the App_tracker project, so
that the user can view and control the analysis of all their activity in one place


Requirements

pip install flask, time

chrome browser that is new enough, the ability to run javascript and python files


Running the Program

Type into cmd:

python track_web.py

or

open the same file in your IDE and run it
